This is a game made for Philosophy 4 Everybody, coded primarily using the Codex feature on ChatGPT.
As part of the first prototype, there are three rankable values. This can be upscaled to a maximum of 25 depending on how testing goes and if any issues show themselves during testing.
To run the program, first download the files onto your device, then run the 'index' file. The 'index' file is contained within the 'what-matters-most' folder. The other three files are supporting the running of the index file.

This game is made to be simple, there are no additional features right now (such as a database or saved results), however this is planned to be added later.
